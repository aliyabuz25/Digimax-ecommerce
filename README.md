# Digimax E-commerce Platform

## Project Overview
This platform is a specialized digital commerce system developed for **Digimax Digital Enterprise**. It features a modern user interface, comprehensive administrative controls, and an optimized experience for digital product distribution.

---

## Design and Development
This system was designed and implemented by **Ali Yabuz** for official delivery to **Digimax**.

**Design Philosophy:**
- Modern UI: High-end aesthetics with premium color schemes.
- Full Responsiveness: Optimized for mobile, tablet, and desktop viewports.
- Performance: Minimalist implementation focusing on speed and intuitive navigation.

---

## Technical Architecture

### Backend & Database
- Node.js & Express: High-performance server-side infrastructure.
- SQLite Engine: Lightweight and efficient database management.
- BCrypt Security: Secure authentication and encryption layers.
- Session Management: Reliable session handling for user isolation and safety.

### Frontend
- EJS Templating: Dynamic server-side rendering for improved SEO and load times.
- Vanilla CSS: Custom design system for unique visual identity without external dependencies.
- Modular Logic: Integrated components for interactive elements.

---

## Key Features

- Product Management: Dynamic listing, stock tracking, and automated pricing displays.
- Admin Dashboard: Detailed sales analytics and user management tools.
- User Profiles: Integrated balance system, order tracking, and account security.
- Smart Cart: Fluid shopping cart updates and streamlined checkout processes.
- Category Control: Customizable layouts and visual category management.

---

## Installation

To run the project in a local environment:

1. Install dependencies:
   ```bash
   npm install
   ```

2. Initialize the database (if required):
   ```bash
   node seed-admin.js
   ```

3. Start development server:
   ```bash
   npm run dev
   ```
