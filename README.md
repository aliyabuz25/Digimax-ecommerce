# 💎 Digimax E-commerce Platform

![Digimax Banner](public/digimax_app_logo_icon.png)

## 🌟 Proje Hakkında
Bu platform, **Digimax Digital Enterprise** için özel olarak geliştirilmiş, yeni nesil bir e-ticaret ve dijital ürün satış sistemidir. Modern arayüzü, gelişmiş yönetim paneli ve kullanıcı dostu deneyimi ile dijital pazaryeri standartlarını belirlemek üzere tasarlanmıştır.

---

> [!IMPORTANT]
> Bu sistem **Ali Yabuz** tarafından tasarlanmış ve kodlanmış olup, **Digimax** şirketine resmi olarak teslim edilmek üzere optimize edilmiştir.

---

## 🛠️ Teknik Özellikler

### **Backend & Veritabanı**
- **Node.js & Express:** Yüksek performanslı sunucu altyapısı.
- **SQLite Engine:** Hafif, hızlı ve taşınabilir veritabanı mimarisi.
- **BCrypt Security:** Güvenli parola şifreleme ve yetkilendirme.
- **Session Management:** Hızlı ve güvenli oturum yönetimi.

### **Frontend**
- **EJS Templating:** Dinamik ve hızlı render edilen sayfalar.
- **Vanilla CSS:** Özel tasarım tokenları ile geliştirilmiş, Tailwind veya Bootstrap bağımlılığı olmayan saf görsel şıklık.
- **Interactive UI:** Vue tabanlı mikro-komponentler ve dinamik animasyonlar.

## 🚀 Temel Fonksiyonlar

- **📦 Ürün Yönetimi:** Detaylı ürün kartları, stok takibi ve dinamik fiyatlandırma.

## 📦 Kurulum

Projeyi yerel ortamınızda çalıştırmak için şu adımları izleyin:

1. Bağımlılıkları yükleyin:
   ```bash
   npm install
   ```

2. Veritabanını hazırlayın (gerekirse):
   ```bash
   node seed-admin.js
   ```

3. Geliştirici modunda çalıştırın:
   ```bash
   npm run dev
   ```
